# 서구 · 부산광역시

> 서구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-busan-seo.github.io/**

서구는 부산 원도심의 서쪽 자락에서 남항과 송도 앞바다를 끼고 남북으로 길게 뻗은 자치구로, 6·25전쟁기 대한민국 임시수도의 심장부였던 곳입니다. 부민동 임시수도 대통령 관저와 정부청사가 자리했던 이 일대는 피란민들이 산비탈을 따라 판잣집을 짓고 살던 산복도로 문화의 원형을 고스란히 간직하고 있습니다. 바다 쪽으로는 1913년 문을 연 국내 최초의 공설 송도해수욕장이 백 년 넘게 부산 사람들의 물놀이터로 사랑받아 왔습니다. 부산대·동아대·고신대 세 대학병원과 부산공동어시장이 좁은 면적 안에 밀집해, 의료와 수산이라는 두 축이 지역 경제를 떠받치는 독특한 도시입니다. 근대의 상흔과 항구의 활기, 해안 절경이 한 구역 안에 겹겹이 쌓여 있어 걷는 재미가 각별한 동네입니다.
